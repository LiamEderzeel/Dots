# Dots

## install

- `git clone --bare git@github.com:LiamEderzeel/Dots.git ~/.dots`
- `source ~/.zshrc`
- `config remote remove origin`
- `config remote add git@github.com:LiamEderzeel/Dots.git`

## usage

- use `config` instead of `git`
- use `configlg` for lazygit
